# ketword_tag_cs.txt 說明(請搭配著看)
## [end_point] 裡面設定 ivr 分項
	- 名稱：
		- 等號左邊是 end point 名稱，取名要像是 ivr_111_2_3, 111 代表分項代號，2 代表這個分項的第二個小分項，3 代表該分項/小分項的第三個規則
		- 只有分項代號是必須，就是要取名成至少像 ivr_111，其他可以忽略
		- 名稱要唯一，就是不可以有兩個 ivr_111 的 end point，可以在 ivr_xxx 後面不斷加 _y_z 來做到唯一
	- 中文描述：
		- 等號右邊到第一個逗點是 中文描述部分，請輸入對應的中文敘述
	- tag：
		- 等號右邊第一個逗點後的是 end point 對應到的 tags，請盡量為每個 end point 設定至少兩個 tags
		- tag 說明請向下看
## [tag] 
	- 名稱：
		- 等號左方是 tag 名稱，命名請用英文，需要多英文詞的部分用底線連結，例如 credit_card
	- 對應詞
		- 等號右邊都是該 tag 的對應詞
